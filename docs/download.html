Установка Bunker Boats

Выберите компоненты для установки:

Скачать исходный код (ZIP) Скачать скрипт установки

Инструкции по установке:

  1. Скачайте исходный код (ZIP) и распакуйте его в любую папку
  2. Скачайте скрипт установки и сохраните его в ту же папку
  3. Запустите скрипт установки (optimize_build.bat)
  4. Дождитесь завершения процесса - будет создан установщик
  5. Запустите созданный файл BunkerBoats-Setup.exe

Альтернативный способ (требуется Node.js):

  1. Убедитесь, что у вас установлен Node.js
  2. Скачайте и распакуйте ZIP с исходным кодом
  3. Откройте командную строку в папке с распакованным кодом
  4. Выполните команду: npm install
  5. После установки зависимостей выполните: npm start

Подробная инструкция доступна в README.md
