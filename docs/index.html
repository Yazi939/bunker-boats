Bunker Boats

Приложение для учета и анализа операций с топливом для водного транспорта

Скачать приложение

Выберите один из следующих способов установки:

Скачать исходный код (ZIP) Скачать скрипт установки

Простая установка:

  1. Скачайте исходный код (кнопка ZIP выше)
  2. Распакуйте архив в любую папку
  3. Скачайте и сохраните скрипт установки в ту же папку
  4. Запустите скрипт установки (optimize_build.bat)
  5. После завершения установки запустите приложение через созданный ярлык

Основные возможности

Учет операций

Добавление и управление операциями покупки и продажи топлива

Аналитика

Анализ прибыльности и маржинальности операций с топливом

Экспорт в Excel

Экспорт данных и отчетов в формат Excel для дальнейшей обработки

Статистика

Подробная статистика по видам топлива с визуализацией

Системные требования

  • Windows: Windows 10 или 11, 2 ГБ ОЗУ, 500 МБ свободного места
  • macOS: macOS 10.13 или новее, 2 ГБ ОЗУ, 500 МБ свободного места

Альтернативные способы установки

Для Windows с Node.js:

  1. Убедитесь, что у вас установлен Node.js (рекомендуется версия 16 или выше)
  2. Скачайте и распакуйте ZIP с исходным кодом
  3. Откройте командную строку в папке с распакованным кодом
  4. Выполните команду: npm install
  5. После установки зависимостей выполните: npm start

Для опытных пользователей:

  1. Клонируйте репозиторий: git clone https://github.com/Yazi939/application.git
  2. Перейдите в папку проекта: cd application
  3. Установите зависимости: npm install
  4. Запустите приложение: npm start
